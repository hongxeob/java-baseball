## 기능
* 1~9 사이의 숫자중 3개를 뽑아 3자리 수를 정한다.
  * [예외] 받은 값이 3자리가 아니다
  * [예외] 받은 값이 숫자가 아니다
  * [예외] 받은 값에 중복이 있다.
  

* 입력한 값에 따라 힌트를 준다.
  * 같은 수가 전혀 없으면 - 포볼 or 낫싱
  * 같은 수가 다른 자리에 있으면 - 볼
  * 같은 수가 같은 자리에 있으면 - 스트라이크
